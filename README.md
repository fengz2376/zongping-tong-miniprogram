# 综评通 - 上海高考综合评价AI助手

上海高考综合评价批次招生信息查询与面试准备的微信小程序。

## 功能特性

- **综评流程时间线** - 实时跟踪报名、初审、高考、出分、志愿填报、面试、录取各阶段
- **个人陈述AI撰写** - 针对目标高校定制，自动去AI化，包含多角度个性化内容
- **面试模拟训练** - 120道高频面试题，支持AI追问与即时反馈
- **综评分数计算** - 综合成绩一键计算，录取概率估算
- **高校资料库** - 11所综评高校详细信息，包含2025年专业组分数线
- **AI智能问答** - 综评政策、面试技巧、高校信息随时问

## 支持的高校

复旦大学、上海交通大学、同济大学、华东师范大学、华东理工大学、东华大学、上海外国语大学、上海财经大学、上海大学、上海中医药大学、浙江大学

## 使用说明

1. 克隆本仓库
2. 使用微信开发者工具导入项目
3. 在「我的」页面配置 DeepSeek API Key（如需AI功能）
4. 开始使用

## 技术栈

- 微信小程序原生开发
- Tailwind CSS 样式
- DeepSeek API（AI功能）

## 数据说明

- 高校分数线数据来源于各校官方公布
- 面试题库基于历年真题整理
- 政策信息参考上海市教育考试院公布内容
